### Install

#### 1. Clone this repo

```
$ git clone https://github.com/h1kkar/shime.git
```

#### 2. Run in your shell:

```
$ make build
# make install
```

### Uninstall

#### Run in your shell:

```
# make uninstall
```

*If you do not possess `make`, then just repeat commands in `MakeFile`, but perform them with caution, I do not bear responsibility in the event of your error.*
